# Fetching Player Data

Learn how to fetch real-time player data from Enka.Network.

## Basic Usage

```javascript
const { Client, EnkaManager } = require('genshin-manager')

const client = new Client()
await client.deploy()

const enkaManager = new EnkaManager()
const playerData = await enkaManager.fetchAll(800802278) // Replace with valid UID

console.log(`Player: ${playerData.playerDetail.nickname}`)
console.log(`Level: ${playerData.playerDetail.level}`)
console.log(`Characters: ${playerData.characterDetails.length}`)
```

## Player Detail

The `playerDetail` object contains:

- `nickname` - Player's display name
- `level` - Adventure Rank
- `worldLevel` - Current World Level
- `signature` - Player's signature
- `profilePicture` - Profile picture information

## Character Details

Each character in `characterDetails` includes:

- Basic information (name, element, rarity)
- Current stats and level
- Equipped weapon
- Equipped artifacts
- Talent levels
- Constellation unlocks

## Event Handling

```javascript
const { EnkaManager, EnkaManagerEvents } = require('genshin-manager')

const enkaManager = new EnkaManager()

enkaManager.on(EnkaManagerEvents.FETCH_COMPLETE, (data) => {
  console.log('Data fetched successfully')
})

enkaManager.on(EnkaManagerEvents.FETCH_ERROR, (error) => {
  console.error('Fetch failed:', error)
})
```
