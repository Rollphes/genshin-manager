# Weapon Information

Access detailed weapon data including stats, refinements, and passive abilities.

## Basic Usage

```javascript
const { Client, Weapon } = require('genshin-manager')

const client = new Client({ defaultLanguage: 'EN' })
await client.deploy()

// Create weapon instance
// Parameters: weaponId, level, isAscended, refinementRank
const weapon = new Weapon(13501, 90, true, 5) // Staff of Homa, Level 90, Ascended, R5
console.log(weapon.name)           // "Staff of Homa"
console.log(weapon.stats)          // Array of weapon stats
console.log(weapon.skillName)      // "Reckless Cinnabar"
```

## Weapon Info

For basic weapon information without stats calculation:

```javascript
const { WeaponInfo } = require('genshin-manager')

const weaponInfo = new WeaponInfo(13501, 90, true, 5)
console.log(weaponInfo.name)
console.log(weaponInfo.rarity)
console.log(weaponInfo.weaponType)
```

## Weapon Refinement

```javascript
const { WeaponRefinement } = require('genshin-manager')

const refinement = new WeaponRefinement(13501, 5)
console.log(refinement.skillName)
console.log(refinement.skillDescription)
console.log(refinement.addProps)
```

## Weapon Ascension

```javascript
const { WeaponAscension } = require('genshin-manager')

const ascension = new WeaponAscension(13501, 6) // Max ascension
console.log(ascension.costItems)    // Materials needed
console.log(ascension.costMora)     // Mora cost
console.log(ascension.unlockMaxLevel) // Max level after ascension
```

## Search Weapons

```javascript
const { WeaponInfo } = require('genshin-manager')

// Get all weapon IDs
const allWeaponIds = WeaponInfo.getAllWeaponIds()

// Filter by type
const polearms = allWeaponIds.filter(id => {
  const weapon = new WeaponInfo(id)
  return weapon.weaponType === 'WEAPON_POLE'
})
```
