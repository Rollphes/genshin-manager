# API Reference

Complete API documentation for Genshin Manager.

## Core Classes

### Client & Managers

| Class | Description |
|-------|-------------|
| `Client` | Main entry point for the library |
| `EnkaManager` | Handles Enka.Network API interactions |
| `NoticeManager` | Manages game announcements |

### Data Models

| Class | Description |
|-------|-------------|
| `Character` | Character with stats at specific level |
| `CharacterInfo` | Basic character information |
| `Weapon` | Weapon with stats at specific level |
| `WeaponInfo` | Basic weapon information |
| `Artifact` | Artifact data and set bonuses |
| `Material` | Game materials and items |
| `Monster` | Enemy data and stats |
| `StatProperty` | Stat properties with values |
| `DailyFarming` | Daily farming domain information |
| `ProfilePicture` | Profile picture information |

### Character Classes

| Class | Description |

| `CharacterConstellation` | Constellation information |
| `CharacterCostume` | Character costume data |
| `CharacterInherentSkill` | Passive talents |
| `CharacterProfile` | Character profile/biography |
| `CharacterSkill` | Active skill information |
| `CharacterSkillAscension` | Talent level-up materials |
| `CharacterStatusManager` | Manages character stats |

### Weapon Classes

## Error Classes

All error classes extend `GenshinManagerError`.

### Base Errors
- `GenshinManagerError` - Base error class
- `ValidationError` - Validation failures
- `NetworkError` - Network issues
- `AssetError` - Asset-related errors

### Specific Errors
- `EnumValidationError`, `FormatValidationError`, `RangeValidationError`, `RequiredFieldError`
- `AssetCorruptedError`, `AssetDownloadFailedError`, `AssetNotFoundError`
- `ImageNotFoundError`, `AudioNotFoundError`
- `NetworkTimeoutError`, `NetworkUnavailableError`
- `EnkaNetworkError`, `EnkaNetworkStatusError`
- `JsonParseError`, `KeyMatchingError`, `LowConfidenceError`
- `ConfigInvalidError`, `ConfigMissingError`
